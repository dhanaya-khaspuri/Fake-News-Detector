# Fake-News-Detector
**PROJECT DETAILS:**
This project is a web application that helps users explore news articles and identify potentially misleading or low-credibility content. It fetches real-time news data and applies basic logic to classify articles based on their credibility.
**PURPOSE:**
The goal of this project is to build an interactive web application that demonstrates the use of APIs, data filtering, and logical analysis to solve a real-world problem — identifying fake or misleading news.
**API USED:**
https://newsapi.org/
**FEATURES IMPLEMENTED:**
-Users can search news articles by keywords
-View article details 
-Credibility indicator
-Filter articles by date and source
-Sort articles by latest or popularity
-Responsive Design
**TECHNOLOGIES USED**
HTML
CSS
JavaScript
Fetch API
**HOW THE PROJECT CAN BE SETUP**
The application determines the credibility of news articles using simple rules:
-Articles from trusted sources are marked as High credibility
-Articles containing suspicious keywords are marked as Low credibility
-Others are marked as Medium credibility
